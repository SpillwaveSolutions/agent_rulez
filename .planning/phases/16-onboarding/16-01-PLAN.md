# Plan 16-01: Onboarding Wizard Foundation + UI

## Objective
Add `onboardingComplete` settings field, `check_binary` Tauri command, and the full multi-step onboarding wizard UI.

## Tasks

### Task 1: Add onboardingComplete to Settings
- Add `onboardingComplete: boolean` to `Settings` interface (default: `false`)
- Add `setOnboardingComplete` action to settingsStore

### Task 2: Add check_binary Tauri command
- New `check_binary` command in `debug.rs` exposing binary detection
- Returns `{ found: boolean, path: string | null }`
- Add TypeScript wrapper + mock in `tauri.ts`

### Task 3: Add generateSampleConfig to tauri.ts
- Hardcode the sample hooks.yaml template in TypeScript
- Use existing `writeConfig()` to write it to `~/.claude/hooks.yaml`
- Add mock that returns immediately

### Task 4: Create OnboardingWizard component
- Multi-step wizard: Welcome → Binary Check → Sample Config → Test Simulation → Complete
- Full-screen dialog overlay with `role="dialog"` and aria-label
- Each step as inline section (not separate components for simplicity)
- Skip button available on all steps
- Wire into App.tsx: show when `!settings.onboardingComplete && isLoaded`
