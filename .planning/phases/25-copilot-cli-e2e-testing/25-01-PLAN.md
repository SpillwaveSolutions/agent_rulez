Create `e2e/lib/copilot_adapter.sh` by following `e2e/lib/gemini_adapter.sh` exactly, with these Copilot-specific changes:

1. **COPILOT_CLI_NAME="copilot"** (exported constant)

2. **copilot_adapter_check()**: Check `command -v copilot` only. Do NOT check for any API key env var (Copilot uses OAuth login, not an API key). Print version on success via `copilot --version`. Return 1 with install instructions if not found.

3. **require_copilot_cli()**: Return 0 if `COPILOT_CLI_AVAILABLE` is 1, else return 77 with skip message.

4. **setup_copilot_hooks(workspace, rulez_binary)**: Write `.github/hooks/rulez.json` (NOT `.gemini/settings.json`). Create `mkdir -p "${workspace}/.github/hooks"`. Resolve rulez_binary to absolute path. Write JSON with this exact structure:
```json
{
  "version": 1,
  "hooks": {
    "preToolUse": [
      {
        "type": "command",
        "bash": "<abs_rulez> copilot hook",
        "powershell": "<abs_rulez> copilot hook",
        "timeoutSec": 10
      }
    ]
  }
}
```
Key differences from Gemini: uses `preToolUse` (not `BeforeTool`), uses `bash`/`powershell` fields (not `command`), uses `timeoutSec` in seconds (not `timeout` in ms), has `version: 1` at top level.

5. **invoke_copilot_headless(workspace, prompt, timeout_secs)**: Run `copilot -p "${prompt}" --allow-all-tools` (NOT `--yolo --output-format json` like Gemini). Use `NO_COLOR=true timeout "${timeout_secs}" copilot ...`. Capture to `${workspace}/copilot-output.txt`. Return 77 on timeout (exit 124). Make the script executable (chmod +x).

Create 3 fixture files under `e2e/fixtures/copilot/` — content is IDENTICAL to `e2e/fixtures/gemini/` counterparts:
